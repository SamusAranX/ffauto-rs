# List of dynamic palettes

These palettes are defined as gradients.\
When referencing one of these, a 256-color palette will be derived from these by default.\
You can change the number of steps by specifying a value for `--palette-steps/-pn`.

See also [PRESET.md](https://github.com/mazznoer/colorgrad-rs/blob/master/PRESET.md).
